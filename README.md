# SEO-агентство — автоматизация

## Описание
Проект для автоматизации SEO-процессов.

## Структура
- `scripts/` — скрипты
- `skills/` — скиллы
- `data/` — данные

## Быстрый старт
1. Установить зависимости
2. Настроить `configs/*.env`
3. Запустить `python skills/<skill>/run.py`

## Команда
- SEO-специалисты
- Python-разработчики
- Data-аналитики
