# Паспорт проектов

## Инструкция по созданию Excel файла

1. Откройте файл "Паспорт проектов.csv" в Excel
2. Используйте "Данные" > "Текст по столбцам" для правильного разделения
3. Сохраните как "Паспорт проектов.xlsx"

## Структура данных

- domain: Домен проекта
- active: Активен ли проект (true/false)  
- status: Статус проекта (production/development/maintenance)
- metrika_id: ID счетчика Яндекс.Метрики
- webmaster: Статус подключения к Яндекс.Вебмастер
- gsc: Статус верификации в Google Search Console
- proxy: Прокси-сервер для работы с проектом
