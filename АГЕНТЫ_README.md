# Реестр агентов

## Инструкция по созданию Excel файла

1. Откройте файл "Агенты.csv" в Excel
2. Используйте "Данные" > "Текст по столбцам" для правильного разделения
3. Сохраните как "Агенты.xlsx"

## Описание колонок

- **Агент**: Название агента
- **Тип**: 
  - cron - запускается по расписанию
  - команда - запускается по команде
  - реактивный - запускается при событии
- **Скилл**: Связанный скилл из папки skills/
- **Частота**: Как часто запускается агент
- **Статус**: Текущий статус агента

## Содержимое

Файл содержит 7 агентов различных типов с указанием связанных скиллов и частоты выполнения.
