# Скилл: Выгрузка Яндекс.Метрика

## Описание
Собирает из Яндекс.Метрики многостраничный Excel: сводку роста (период к периоду), поведение по страницам, длину сессий, источники, устройства, географию, поисковые фразы, помесячный тренд и цели. Выгрузка **полная** (без лимитов, постранично).

## Как получить OAuth-токен (один раз)

Метрика использует тот же Яндекс OAuth, что и Вебмастер — **можно использовать один токен**, если при создании приложения выбрать оба доступа.

1. Открой [oauth.yandex.ru/client/new](https://oauth.yandex.ru/client/new): платформа **«Веб-сервисы»**, Redirect URI `https://oauth.yandex.ru/verification_code`.
2. **Доступы:** начни печатать `метрика` и выбери доступ к **API Яндекс.Метрики (получение статистики)**. *(Если нужен и Вебмастер — добавь ещё `webmaster:hostinfo`.)*
3. Создай приложение → получишь **ClientID**.
4. Получи токен: открой `https://oauth.yandex.ru/authorize?response_type=token&client_id=ТВОЙ_CLIENT_ID` и скопируй `access_token` из адресной строки.
5. Проверь токен и заодно посмотри свои счётчики:
   ```bash
   curl -H "Authorization: OAuth ТВОЙ_ТОКЕН" "https://api-metrika.yandex.net/management/v1/counters?per_page=100"
   ```

Сохрани токен в **`configs/metrica/.env`** (в Git не коммитим):
```
YANDEX_METRICA_TOKEN=y0_твой_токен
```
> `counter_id` здесь НЕ храним — он per-project и живёт в Паспорте проектов, передаётся через `--counter-id`.

## Запуск
```bash
# Мок (заглушки, без API)
python run.py --counter-id 12345678

# Реальный API за период (по умолчанию 30 дней)
python run.py --counter-id 12345678 --real
python run.py --counter-id 12345678 --days 90 --real
```

## Выход
`data/metrica/metrica.xlsx` — вкладки:
- **Сводка** — текущий период vs предыдущий (визиты, посетители, просмотры, **органика отдельно**, цели, отказы, ср. время) с Δ% и подсветкой рост/падение
- **По URL** — страницы: просмотры, посетители, отказы, время
- **Длина сессий** — по страницам: доля визитов дольше 1 мин и 3 мин
- **Источники** · **Устройства** · **География** · **Запросы**
- **По месяцам** — помесячный тренд
- **Цели** — достижения по дням

Отказы > 70% подсвечиваются красным на всех листах, где есть колонка отказов.

## Структура
- `run.py` — оркестратор (читает конфиг, собирает вкладки, экспортирует)
- `metrica_client.py` — запросы к API + мок-режим (`use_real`)
- `exporter.py` — Excel с форматированием
