# Скилл: Выгрузка Google Search Console

## Описание
Выгружает из GSC все доступные через API отчёты и складывает их вкладками в один Excel.

Вкладки: **Сводка** (итоги периода), **Запросы**, **Страницы**, **Запрос+Страница**,
**Динамика** (по датам), **Устройства**, **Страны**, **Вид в выдаче**, **Sitemaps**.

## Установка
```bash
pip install pandas openpyxl google-api-python-client google-auth google-auth-oauthlib
```

## Авторизация
Креденшалы лежат в `configs/gsc/` (в Git не коммитим). Поддерживаются два варианта:
- `token.json` — OAuth-токен пользователя (получается через `python authorize.py`,
  для этого нужен `configs/gsc/client_secret.json` — OAuth-клиент типа Desktop)
- `service_account.json` — Service Account (его email должен быть добавлен в пользователи ресурса в GSC)

Проверить, какие сайты доступны:
```bash
python run.py --list-sites
```

## Запуск
```bash
# Мок-режим (генерирует случайные данные)
python run.py --site https://example.com --start 2026-06-01 --end 2026-06-30

# Реальный API
python run.py --site https://example.com/ --start 2026-06-01 --end 2026-06-30 --real
```

> Формат `--site` должен совпадать с тем, как ресурс добавлен в GSC:
> `sc-domain:site.ru` (доменный ресурс) или `https://site.ru/` (URL-prefix).
> GSC хранит данные ~16 месяцев, свежие данные появляются с задержкой 1-2 дня.

## Выход
`data/gsc/<алиас-домена>.xlsx` — например `example-com.xlsx`.
Имя файла строится из `--site` автоматически; переопределить можно через `--output`.

CTR подсвечен цветом: зелёный > 3%, жёлтый 1-3%, красный < 1%.

## Нюансы данных GSC
- Сумма кликов по вкладке «Запросы» может быть меньше «Динамики»: Google скрывает
  анонимизированные (редкие) запросы из разреза по запросам, но учитывает их в итогах.
- «Вид в выдаче» часто пустой — вкладка есть, если у сайта нет спец-элементов выдачи.
- Отчёты «Покрытие» и Core Web Vitals через API не отдаются (только URL Inspection поштучно).

## Структура
- `run.py` — оркестратор, сводный лист, алиас домена для имени файла
- `gsc_client.py` — авторизация, все отчёты Search Console API v1 (пагинация по 25 000 строк)
- `exporter.py` — Excel: вкладка на отчёт, шапка, автоширина, подсветка CTR
- `authorize.py` — одноразовая OAuth-авторизация аккаунта (создаёт token.json)
